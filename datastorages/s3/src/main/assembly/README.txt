CMIS data storage provider library
----------------------------------

1) Copy the provided jar files to geonetwork/WEB-INF/lib folder

2) Define the following environmental variables

Use S3 data storage provider:

export GEONETWORK_STORE_TYPE=s3

Setup S3 connection: https://geonetwork-opensource.org/manuals/4.0.x/en/install-guide/customizing-data-directory.html#using-a-s3-object-storage

3) Start GeoNetwork

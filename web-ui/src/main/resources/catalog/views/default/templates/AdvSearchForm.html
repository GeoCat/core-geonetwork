Advanced Search Tips


Title / Keywords / eCatIDs

Title: Search phrase results records that contain any word within the title .

Keywords:To find more keyword list, start typing the keyword.

eCatIDs: For more eCat records, provide eCatIDs seperated by comma.


Author

Author: Type atleast one character to get the author names.

Owner: This predictive search provide the owner who created the record.


Status / Kind

This search is only for internal GA users.

Status: Search records based on the workflow status of metadata.

Kind: This option provides the user to select between metadata and template.


Creation Date / Publication Date / Temporal Extent Period

To search for a Records created or Published or temporal extent period within a date range choose the 'From' and 'To' option. Use predefined date range by selecting the caret. Options: Today/Yesterday/This week/This month/Last 3 months/Last 6 months/This year.
